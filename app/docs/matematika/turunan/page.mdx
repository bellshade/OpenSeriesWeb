# Derivative (Turunan)

Turunan Merupakan konsep Kalkulus yang mempelajari bagaimana perubahan
nilai pada suatu titik.

Rumus:

```
\lim_{{x \to a}} f(x)
```

Fungsi [sumber kode [disini](https://github.com/bellshade/OpenSeries/blob/8766497be42a76a175a94a84cc5609c94c015c61/OpenSeries/matematika.py#L552)]

```python
def turunan(f: Callable[[float], float], x: Union[int, float]) -> float:
```

Contoh kode

```python
import OpenSeries.matematika as matematika

def f(x):
  return x**2
nilai_x = 4
print(matematika.turunan(f,nilai_x))
```
