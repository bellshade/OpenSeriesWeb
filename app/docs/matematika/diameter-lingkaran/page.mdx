# Diameter Lingkaran

Diameter lingkaran adalah jarak antara dua titik pada lingkaran yang melalui titik pusat. Diameter juga merupakan dua kali panjang jari-jari lingkaran. Rumus diameter lingkaran adalah:

```
d = 2r
```

Keterangan:

- d adalah diameter lingkaran
- r adalah jari-jari lingkaran

Rumus diameter lingkaran ini biasa digunakan untuk menghitung diameter jika diketahui jari-jari, atau sebaliknya.

