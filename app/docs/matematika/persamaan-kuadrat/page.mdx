# Persamaan Kuadrat

Persamaan kuadrat adalah persamaan yang memiliki variabel dengan pangkat tertinggi dua. Contohnya, x2+4x+6=0 adalah persamaan kuadrat.
Rumus persamaan kuadrat yang paling umum adalah:

```
ax² + bx + c = 0
```

`dengan a,b,c adalah konstanta dan a≠0.`


Rumus ini biasa digunakan untuk mencari akar-akar persamaan kuadrat, yaitu nilai-nilai x yang memenuhi persamaan tersebut. Ada beberapa cara untuk mencari akar-akar persamaan kuadrat, seperti faktorisasi, melengkapi kuadrat sempurna, atau rumus ABC.
